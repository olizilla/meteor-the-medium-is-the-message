What is this Meteor?

A full stack framework

Manages client and server side

Reactive by default

Your UI updates as the db changes, out of the box.

KISS

Meteor keeps it simple...

Getting started is simple

You're 4 commands away from your first Meteor app.


  curl https://install.meteor.com | /bin/sh
	
  meteor create the-medium-is-the-message
	
  cd the-medium-is-the-message

  meteor

Batteries Included

  • CSS & JS munged, cached & versioned
  • Hot code reload; save a file, see a change

Sharing data is simple

Add to a collection; other users see the change


 var Things = new Meteor.Collection('things')
 Things.insert({ what: 'awesome' })

Reactivity is simple

Templates re-render when the collection changes


 Template.awesomeThings.all = function () {
   return Things.find({ what: 'awesome' })
 }

Authentication is simple

	
  meteor add accounts-twitter
  meteor add accounts-facebook
  meteor add accounts-ui

Deploying is simple

	
  meteor deploy goto
  open http://goto.meteor.com
